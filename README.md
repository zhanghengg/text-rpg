# Mist Ring: Text RPG

A turn-based text RPG built with Next.js App Router.

- Local save (browser `localStorage`)
- Jobs: Guard, Ranger, Warlock, Cleric, Rogue, Scholar
- Maps (planned): Borderlands, Mistwood, Old Mine, Rift Corridor

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
